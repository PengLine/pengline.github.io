配置文件的权限修改

config 、id_rsa_github和 id_rsa_gitee 配置文件需要删除其他用户的权限，只保留管理员和当前用户

  1. 右键需配置的文件，查看属性，打开“安全”一栏
  2. 打开“高级” ，“禁用继承”
  3. 返回，在“安全”一栏的“组或用户名” 中，删除非当前用户和系统管理员的账号，如：Everyone
