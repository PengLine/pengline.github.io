API文档

使用我们提供的RESTful API，您可以轻松集成视频处理功能到您的应用程序中。

当前版本

v1.0.0

返回首页

API端点

视频处理

上传视频文件并指定处理需求，获取处理后的视频。

POST
/api/process-video
请求参数
参数名 类型 是否必须 描述
query String 是 视频处理需求的文字描述
files[] File 是 要处理的视频文件，支持多个文件，格式为MP4、AVI、MOV、MKV
成功响应
{"status": "success", "message": "视频处理成功", "video_url": "/api/video/xxx.mp4", "report": {...}}
错误响应
{"status": "error", "message": "错误描述"}

视频下载

获取处理后的视频文件。

GET
/api/video/<filename>
路径参数
参数名 类型 描述
filename String 视频文件名
响应

返回视频文件流

健康检查

检查服务是否正常运行。

GET
/api/health
响应
{"status": "healthy", "agents_loaded": 4}

示例代码

Python

import requests

# 视频处理
url = "http://localhost:8000/api/process-video"
files = {'files[]': open('video.mp4', 'rb')}
data = {'query': '提取所有包含猫的片段并按时间顺序排列'}

response = requests.post(url, files=files, data=data)
result = response.json()

if result['status'] == 'success':
    print('处理成功!')
    video_url = result['video_url']
    # 下载视频
    video_response = requests.get(video_url)
    with open('output.mp4', 'wb') as f:
        f.write(video_response.content)
else:
    print('处理失败:', result['message'])

JavaScript (Fetch API)

// 创建FormData对象
const formData = new FormData();
formData.append('query', '提取所有包含猫的片段并按时间顺序排列');
formData.append('files[]', fileInput.files[0]);

// 发送请求
fetch('http://localhost:8000/api/process-video', {
    method: 'POST',
    body: formData
})
.then(response => response.json())
.then(data => {
    if (data.status === 'success') {
        console.log('处理成功!');
        const videoUrl = data.video_url;
        // 显示或下载视频
        const videoElement = document.createElement('video');
        videoElement.src = videoUrl;
        videoElement.controls = true;
        document.body.appendChild(videoElement);
    } else {
        console.error('处理失败:', data.message);
    }
})
.catch(error => {
    console.error('请求出错:', error);
});

最佳实践

  • 详细描述处理需求

    提供具体、详细的处理需求描述，以获得更准确的结果。

  • 合理控制视频大小

    过大的视频文件可能导致处理时间过长，请考虑预先压缩视频。

  • 错误处理

    始终检查API响应的status字段，并妥善处理可能出现的错误。

  • 异步处理

    对于大型视频处理任务，考虑使用异步处理模式，避免长时间阻塞请求。
