关于我

个人简介

资深软件架构师，8年 Java 高并发系统设计经验，近一年深度投入 AIGC 领域，主导短剧内容生成平台的技术架构与 AI 模型工程化落地。精通 Spring Cloud、微服务架构、分布式系统性能优化，同时具备扎实的 AI 工程能力，熟练使用 Stable Diffusion、LoRA、ComfyUI、vLLM、Dify 等工具构建文生图、图生图、AI 视频生成流水线。擅长将 AI 能力与企业级系统融合，打造“数据驱动 + 智能生成”的下一代内容平台。具备从底层架构到上层 AI 应用的全栈交付能力。

专业技能

  • 架构设计：基于高内聚低耦合，实现高并发、高可用和可扩展设计，擅长微服务拆分（DDD领域）、服务治理、CQRS模式，主导过单体系统到分布式架构的重构升级。

  • Java 与 分布式：深入掌握 Java EE 核心机制（JVM、多线程、GC调优），熟练 Spring Boot、Spring Cloud Alibaba 构建分布式系统。规则引擎LiteFlow实现解耦和组件编排，Spring 状态机增加扩展，Seata分布式事务解决方案。

  • 数据层架构：

    • 熟练MySQL 性能优化与分库分表设计，基于 ShardingSphere 的亿级数据水平拆分方案，支持线性扩展
    • 熟练运用 Redis 实现缓存穿透/雪崩防护、登录鉴权、分布式锁、位统计，提升核心接口 QPS 至15,000+
    • 熟悉 Elasticsearch 在海量数据中的复杂查询与实时检索应用。解决深度分页查询和 ELK 日志收集与分析

  • 中间件与异步解耦：熟悉 RocketMq/RabbitMq 中间件，设计消息解耦的IOT系统，保障系统可靠性与削峰能力。

  • DevOps 与云原生：熟悉 Docker、Kubernetes 容器化部署，通过 Jenkins 搭建 CI/CD 流水线实现自动化发布，监控体系集成 Prometheus + Grafana + SkyWalking。

  • 性能优化：全链路性能优化经验，从 JVM → SQL → 缓存 → 异步多维度提升系统吞吐量，平均响应时间下降 75% 。

  • AI 与 LLMOps：

    • 了解主流 AI 框架，参与过基于 SD 和 Flux 扩散模型的图像生成类项目，基于 LoRA 模型实现特定领域微调
    • 了解MCP、A2A协议，使用 LangChain、MetaGPT 构建智能体，vLLM 推理和量化，GraphRAG 数据增强
    • 使用 Dify 构建 AI 应用，通过 ComfyUI 与 Java 系统集成，实现自动生图、换装。Airflow 编排任务批处理

  • 大数据技术栈：

    • 使用 Hadoop、Spark（Spark SQL）、Kafka + Flink 构建数据处理流水线，实现设备告警与设备状态分析
    • 了解 Flink、Hive、HBase 等组件，了解 ClickHouse、Doris 等列式 OLAP 数据库，实现实时联机分析处理

  • IoT 与边缘计算：

    • 熟悉 MQTT、CoAP 等协议，对接三方厂商设备、物联网开放平台，实现设备状态监控、远程控制等功能
    • 设计边缘网关数据采集处理模块（Java + MQ），支持万级设备并发接入，支持1500+ TPS 并发数据处理
    • 了解基于 C 语言的 FPGA、PLC、Arduino、Stm32 编程开发，使用传感器 + 单片机 + 串口调试数据写入

领域经验

  • 主导“智慧城管一体化平台”的整体架构设计与技术实施，构建基于 物联网 + 云计算 + 大数据 + GIS 可视化 的城市智能治理体系，覆盖街道管理、环卫监管、市政设施、排水监测、智慧停车、智能照明六大核心场景。

  • 设计“端—边—云”协同的微服务架构：前端设备（传感器、摄像头）通过边缘网关实现协议转换与数据预处理，云端基于 Spring Cloud Alibaba 构建中台系统，实现设备接入、事件中心、预警引擎、工单调度等模块解耦与高可用部署。

  • 搭建统一物联网平台，接入超 50 w 台设备（包括液位计、传感器、GPS终端、智能果壳箱等），采用 MQTT + Kafka 多种协议实现高并发数据采集，日均处理设备消息达百万条，支持设备状态实时监控与远程控制。

  • 构建实时数据处理流水线：基于 Kafka + Flink 实现街道异常事件识别（如垃圾堆积、占道经营）、排水井液位预警、路灯故障检测等场景的毫秒级响应，关键预警平均延迟 < 3 秒。

  • 深度集成 GIS 地理信息系统（SuperMap + Mapbox），打造“一图统管”可视化大屏，实现设备分布、作业轨迹、风险热力、停车位状态等多维数据动态呈现，支撑领导决策与应急指挥。

  • 推动城市管理流程闭环化：实现“自动监测 → 智能预警 → 工单派发 → 现场处置 → 结果反馈 → 效果评估”全流程数字化，事件平均响应时间从 4 小时缩短至 45 分钟，问题闭环率达 93%。

  • 带领 12 人跨职能团队（前端、后端、AI、GIS），采用敏捷+原型开发模式，按季度迭代功能模块，确保项目按期交付并持续优化。

  • 系统上线后，城市管理事件主动发现率提升 300%，人工巡查成本降低 50%，内涝预警提前 30 分钟以上，智慧停车覆盖率提升至 85%，项目入选“省级别数字化改革优秀案例”。
