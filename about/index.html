标杆项目实践

AIGC 创意工坊（Agent）

多模态智能体

端到端 AI 内容创作平台，整合智能调度、多模态生成与视频剪辑，打通“创作→分析→生成→剪辑”自动化生产闭环，实现从用户指令到最终成片的无人化流水线。

NeoTask Flask/FastAPI ComfyUI/SD/Flux Qwen3/Deepseek/Huggingface/Ollama Langchain/LangGraph/Llamalndex MCP/A2A Embedding/Chroma FFmpeg/OpenCV RAG 知识库/召回策略 分布式任务调度/去中心化 一致性策略/状态快照/连续性锚点 视频合成/渲染/配音 文生图、图生图、文生视频、文生音频
  • 自研 NeoTask 分布式调度：支持优先级队列/DAG工作流/自动重试，为高耗时 AI 推理任务提供高可靠异步处理基座
  • 剧本分镜 Agent：多智能体协作 + 向量记忆机制，解析任意格式剧本生成适配 Sora/Runway 的连贯提示词，彻底解决跨片段角色一致性难题
  • 视频混剪 Agent：需求/内容/编辑/质量多智能体协同，基于自然语言描述与视觉分析，自动完成素材智能筛选、裁剪与合成

短剧创作平台（AIGC）

数字资产生态

面向 C 端的“一键生成”内容平台，集成虚拟角色合拍、LoRA 低代码训练与数字资产交易。打造“创作-分享-变现”生态，大幅降低多模态内容生产门槛。

DifyvLLM Qwen3/DeepSeekComfyUI/SD/Flux Lora LiteFlow/Airflow Stable Video Diffusion/Wan2 Docker 可灵/海螺 Spring Cloud Alibaba SkyWalking/Prometheus 虚拟交友 数字资产 模型训练（扩散模型）
  • LoRA 低代码训练平台：用户上传照片自动完成清洗/标注/训练/评估，可视化对比生成效果并一键发布至模型市场
  • 架构高扩展设计：责任链自动解析 ComfyUI 节点；工厂+桥接模式实现多 AI 创作动态切换；模板方法+装饰器增强剪辑 API
  • 异步调度体系：MQ 构建多租户异步任务排队与消峰，Airflow 批量调度图像 CDN 加速分发，保障高并发生成稳定性

物流仓储运输系统（TMS）

分布式架构

原单一 MySQL 库（800+表/单表1亿+）遭遇严重性能瓶颈。基于 DDD 平滑重构至分布式微服务，解决海量数据存储、高并发写入与跨库复杂查询难题。

Spring Cloud AlibabaShardingSphereCanal+ESSeata ATSkyWalking RabbitMQ/Redis Drools/DDD SkyWalking/Prometheus 性能优化 架构升级 分库分表
  • 架构重构与治理：DDD 拆分为 12 个微服务模块，Seata AT 保障分布式事务一致性，Sentinel 实现全链路限流熔断与自适应保护
  • 分库分表实战：user_id+order_id 复合分片（4库×16表），Canal 监听 binlog→MQ→ES 构建二级索引，跨库查询 15s → <2s
  • 性能与可用性跃升：系统可用性 95% → 99.95%，订单创建 >3s → <200ms，核心 QPS 突破 15,000+，日处理订单 10万 → 20万+

智慧城市管理平台（AIOT）

IoT + 数字孪生

整合公安/政务多系统的数据兼容，构建城市全要素数字化监管平台。突破异构设备接入、高并发空间计算与多源协议差异挑战，实现城市治理从“被动响应”向“主动预警”的范式升级。

Flink CDC 边缘网关/OneNET GIS/PostGIS/JTS 微服务中台 AI视频分析 MQTT/CoAP Kafka/RabbitMQ WebSocket/Dubbo 实时预警 车辆GPS监控 水位水压监控 设备状态监控 工地/厂区巡查 事件-上报-派遣-核实-处理-完结-归档
  • 攻克多厂家协议差异与数据幂等性，稳定接入 50万+ 异构设备，实时处理 1000+ QPS 空间坐标计算与预警
  • Flink 实时引擎 + GIS“一图统管”联动，空间轨迹计算与告警响应从 4h 缩短至 45min
  • 主动发现率 ↑300%，问题闭环率达 93%，运维成本 ↓50%，成功孵化“智慧街面/环卫”多城基线产品

面向B端的核心安防基线产品，深度集成安检机、安检门等10+类硬件协议。构建“视频监控+智能安检”立体安防体系，支持人包匹配、包裹判图、客流预警与温度异常检测。

  • 单机承载 500+安检机/1000+安检门 接入，稳定处理 500+ QPS 消息并发与千万级数据分片存储
  • 运用策略/模板等设计模式重构人包关联与双视角绑定逻辑，消除硬编码，显著提升组件拓展性与维护效率
  • 主导数据中台组件迭代与安全加固，输出标准化 UML 架构设计与 FAQ 运维手册，支撑多定制项目高效交付

一站式智能 ERP 商业平台，覆盖采购商、供应商、服务商等多角色协同。集成智能设备监控、商品竞价交易、安装维保与工程结算全链路，为企业提供闭环管理解决方案。

SpringHibernateMySQLRedisjQuery/Bootstrap Nginx JSP 设备维护管理 采购/竞价询价 供应商/采购商 订单校验 保时捷车站大屏 物料-生产-销售
  • 独立负责询价、采购与设备资源模块全栈开发，实现多角色竞价报价、订单流转与购物车交易闭环
  • 完成灯箱设备调度 APP 与保时捷大屏展示系统开发联调，主导驻场部署安装与最终验收交付
