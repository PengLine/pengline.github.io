简介

Claude Code 是 Anthropic 推出的官方命令行 AI 编程助手。它运行在你的终端中，可以直接读写文件、执行命令、搜索代码库，帮助你完成各种软件工程任务。

核心能力

  • 文件操作：读取、创建、编辑文件，支持图片和 PDF

  • 代码搜索：基于 ripgrep 的正则搜索、Glob 模式匹配

  • 命令执行：在沙箱中执行 Bash 命令

  • Git 集成：提交、分支管理、PR 创建与审查

  • 并行处理：通过子代理（Subagent）并行执行多个独立任务

  • 任务追踪：结构化的待办事项管理

  • 记忆系统：跨会话持久化偏好与上下文

  • 网页访问：WebFetch（抓取网页）和 WebSearch（搜索）

参数 说明

配置系统

配置优先级（从高到低）

  1. CLI 参数 — --model、--dangerously-skip-permissions 等

  2. 环境变量 — ANTHROPIC_AUTH_TOKEN、ANTHROPIC_BASE_URL 等

  4. 项目设置 — <project>/.claude/settings.json

  5. 全局用户设置 — ~/.claude/settings.json

  6. 组织设置 — 管理员从 Anthropic Console 推送

  7. 托管设置 — managed-settings.json（管理员强制策略）

权限与安全

权限模式

模式 说明
Ask（默认） Claude 在执行每个操作前询问
Accept Edits 允许自由编辑文件，命令仍需确认
Bypass Permissions 完全信任模式
Auto 模型自行判断操作安全性（需要特定用户资格）

权限规则语法

沙箱安全

  • Linux：PID 命名空间隔离

  • 网络：可配置 sandbox.network.allowedDomains / deniedDomains

  • 文件系统：可配置 sandbox.filesystem.allowWrite / denyRead

  • 自动识别：只读命令（如 ls、grep）自动跳过权限提示

  • 超时控制：每个命令可设置执行超时

安全建议

  • 不要在公共项目中提交 .claude/settings.json 中的敏感信息

  • 使用 .claude/settings.local.json 存放个人配置（加入 .gitignore）

  • 仅在你完全控制的终端中使用 --dangerously-skip-permissions

  • 定期使用 /less-permission-prompts 优化权限规则

Plan 模式与任务追踪

Plan 模式

进入计划模式，Claude 将仅探索和分析，不执行任何更改：

工作流程：

  1. 描述你的需求

  2. Claude 探索代码库，设计方案

  3. 审查计划，提出修改意见

  4. 确认后，Claude 开始实施

  5. 完成后标记任务完成

任务追踪

Claude 会自动创建结构化任务列表：

  • ✅ 已完成 — 绿色标记

  • 🔄 进行中 — 带旋转动画

  • ⏳ 等待中 — 灰色标记

  • 🔒 阻塞 — 红色标记（依赖其他任务）

最佳实践

项目初始化

  1. 首次使用在新项目中运行 /init 创建 CLAUDE.md

  2. 配置 .claude/settings.local.json（添加到 .gitignore）

  3. 用 /statusline 设置有用的会话信息展示

会话管理

  1. 切换任务时用 /rename 给会话取描述性名称

  2. 开启新任务时用 /clear 清空上下文

  3. 长时间会话用 /compact 减少 token 消耗

  4. 用 /branch 分叉会话探索替代方案

  5. 出错时用 /undo 撤销

效率技巧

  1. 用 @文件名 引用文件（支持 Tab 补全）

  2. 批量相关修改，而非一次一个

  3. 使用 /loop 自动化周期性任务

  4. 明确描述需求，Claude 表现更好

  5. 不要模糊，给具体路径和场景

安全性

  1. 仔细审查权限提示，不要盲目批准

  2. 仅在完全可信的环境中使用 --dangerously-skip-permissions

  3. 用 /less-permission-prompts 分析和优化权限列表

  4. 敏感文件（.env、密钥）放在沙箱 denyRead 中

  5. 定期检查 .claude/settings.json 中的安全设置
