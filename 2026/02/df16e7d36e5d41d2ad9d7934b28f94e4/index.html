API参考

核心类

PenshotFunction

参数 类型 默认值 说明
language ShotLanguage ZH 语言
max_concurrent int 10 最大并发数

方法

方法 说明
breakdown_script() 同步调用，等待完成
breakdown_script_async() 异步提交，返回task_id
get_task_status() 获取任务状态
get_task_result() 获取任务结果
wait_for_result() 等待任务完成

ShotConfig

参数 类型 默认值 说明
model_name str “gpt-4” 模型名称
base_url str - API地址
api_key str - API密钥
temperature float 0.7 温度参数
max_tokens int 8192 最大输出token

数据模型

PenshotResult

字段 类型 说明
task_id str 任务ID
success bool 是否成功
status str 任务状态
data Dict 结果数据
error Optional[str] 错误信息
processing_time_ms int 处理耗时

任务状态

状态 说明
PENDING 等待处理
PROCESSING 处理中
COMPLETED 已完成
FAILED 失败

输出结构

错误处理

HTTP状态码 说明
200 成功
400 请求参数错误
404 任务不存在
500 服务器内部错误

常见问题

问题 解决方案
连接失败 检查MCP Server是否运行
任务超时 增加timeout或使用异步模式
结果为空 检查剧本格式是否正确
队列已满 增加queue_size或等待

配置建议

场景 任务池大小 记忆类型 窗口大小 向量库
短剧本(<1000字) 并发5 缓冲记忆 4000 可选
中剧本(1000-5000字) 并发10 摘要记忆 8000 推荐
长剧本(>5000字) 并发20 向量记忆 12000 必需
批量处理 并发50 无 按需 推荐
