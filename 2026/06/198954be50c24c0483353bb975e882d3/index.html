基于 OpenSearch + 文本向量的推荐系统

发表于2026-06-09|更新于2026-07-04
|架构推荐系统
|浏览量:

阶段一之文本向量召回

  • 推荐系统架构总览（系统总体架构、多路召回、混合检索、多模态融合、重排策略）
  • 阶段一：基于文本向量（文本向量化、BERT推理、双路召回、规则排序、存储架构）
  • 阶段二：基于多模态融合（三模态特征提取、动态权重融合、粗排层、GBDT精排）
  • 阶段三：基于个性化精排（实时行为流、滑动窗口向量、DIN模型、自适应E&E）
  • 各阶段算法策略（召回、粗排、精排、重排，协同过滤、向量双塔、热门召回、MMoE与PLE、线性加权、滑动窗口、MMR）

阶段定位与目标

文本向量召回是整个推荐系统的“地基”。在娱乐视频场景中，阶段一的目标不是追求极致的推荐效果，而是用最小的成本快速跑通“视频 → 向量 → 检索 → 推荐”的完整链路，验证技术选型的可行性，并为后续多模态扩展奠定基础。

功能完整 支持I2I/U2I双路召回 两路召回均可独立返回结果
性能 召回延迟 P99 < 100ms
容量 支持视频量级 1000万条视频索引
准确率 召回相关性 人工评估 > 60%
可用性 服务可用性 99.9%

  • 仅使用文本特征：标题、描述、标签，不涉及视觉/音频

  • 离线向量化：文本向量在视频入库时离线生成，不涉及实时计算

  • 简单用户画像：用户向量基于历史观看视频的向量平均，按天更新

  • 轻量精排：阶段一不做复杂模型，按召回分数和简单规则排序

关于粗排与精排的说明：阶段一为了快速上线，直接使用规则排序（轻量级打分）作为排序环节，未严格区分粗排和精排两层的架构。在阶段二的完整漏斗中，召回→粗排→精排→重排是标准的分层架构。阶段一的“规则排序”实际承担了粗排+精排的合并职能，以降低初期实现复杂度。

用户向量计算规则：

行为类型 权重 说明
完播 1.0 完整观看，强正向信号
点赞 0.8 明确喜好
分享 1.2 强烈推荐意愿
评论 0.5 有一定参与度
跳过（前5秒退出） -0.3 负向信号，不加入向量

$$ 用户向量 = \frac{Σ (行为权重 × 视频向量)} { Σ 行为权重} $$

最终向量 = L2_Normalize(用户向量)

规则排序设计

阶段一不引入复杂的CTR模型，采用轻量级规则排序策略：

$$ finalScore = recallScore × α + freshnesScore × β + qualityScore × γ $$

因子 计算方式 权重 说明
recall_score 融合后的召回分数 0.6 主要相关性信号
freshness_score 1 / (1 + age_hours/24) 0.2 24小时半衰期
quality_score log(play_count+1) / 10 0.2 质量信号，限制上限1.0

阶段一规则排序承担的角色：在阶段二的完整漏斗中，规则排序被拆分为“粗排（轻量级DNN）+精排（GBDT）”两层。阶段一将两者合并，是为了以最小成本快速验证召回链路。

规则排序性能指标：

指标 目标值 说明
排序P99延迟 ≤30ms 轻量级计算
排序吞吐 ≥5000 QPS 无模型推理

用户向量计算模型

阶段一的用户画像采用简单高效的加权平均模型，不涉及深度学习。

graph LR
    subgraph "输入层"
        B1[行为1

        B2[行为2
视频向量v2
权重w2]
        B3[...]
        Bn[行为n
视频向量vn
权重wn]
    end
    
    subgraph "加权层"
        W1[v1×w1×t1]
        W2[v2×w2×t2]
        W3[...]
        Wn[vn×wn×tn]
    end
    
    subgraph "聚合层"
        Sum[向量求和]
        Norm[L2归一化]
    end
    
    subgraph "输出"
        U[用户向量
768维]
    end
    
    B1 --> W1
    B2 --> W2
    Bn --> Wn
    W1 --> Sum
    W2 --> Sum
    Wn --> Sum
    Sum --> Norm --> U

时间衰减函数： $$ timeDecay = e^{(-Δt / τ)} $$

Δt: 行为发生距今天数

τ: 半衰期参数，阶段一取7天

行为权重配置：

行为 基础权重 衰减系数 说明
完播 1.0 0.95 完整观看，强信号
点赞 0.8 0.90 明确喜好
分享 1.2 0.85 强烈推荐意愿
评论 0.5 0.92 有一定参与度
关注 1.1 0.88 对创作者认可
重播 0.7 0.93 重复消费价值
