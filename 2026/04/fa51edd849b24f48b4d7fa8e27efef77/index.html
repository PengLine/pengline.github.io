TaskPoolConfig

任务池配置类。

存储配置

参数 类型 默认值 说明
storage_type str "memory" 存储类型：memory/sqlite/redis
sqlite_path str "neotask.db" SQLite 数据库文件路径
redis_url str None Redis 连接 URL

执行器配置

参数 类型 默认值 说明
executor_type str "async" 执行器类型：async/thread/process
max_workers int 10 最大工作线程数

Worker 配置

参数 类型 默认值 说明
worker_concurrency int 10 并发 Worker 数量
prefetch_size int 20 预取任务数量
task_timeout float None 任务超时时间（秒）

队列配置

参数 类型 默认值 说明
queue_max_size int 10000 队列最大长度
priority_levels int 4 优先级级别数

锁配置

参数 类型 默认值 说明
lock_type str "memory" 锁类型：memory/redis
lock_timeout int 30 锁超时时间（秒）

重试配置

参数 类型 默认值 说明
max_retries int 3 最大重试次数
retry_delay float 1.0 重试延迟（秒）

监控配置

参数 类型 默认值 说明
enable_metrics bool True 启用指标收集
enable_health_check bool True 启用健康检查
enable_reporter bool False 启用上报器

节点标识

参数 类型 默认值 说明
node_id str socket.gethostname() 节点唯一标识

SchedulerConfig

调度器配置类。

存储配置

参数 类型 默认值 说明
storage_type str "memory" 存储类型
sqlite_path str "neotask.db" SQLite 路径
redis_url str None Redis URL

Worker 配置

参数 类型 默认值 说明
worker_concurrency int 10 并发 Worker 数

重试配置

参数 类型 默认值 说明
max_retries int 3 最大重试次数
retry_delay float 1.0 重试延迟（秒）

调度配置

参数 类型 默认值 说明
enable_persistence bool False 启用周期任务持久化
scan_interval float 0.05 调度循环扫描间隔（秒）

存储配置详解

内存存储（开发测试）

特点：

  • 速度快，无 I/O 开销

  • 适合开发和单元测试

  • 进程重启后任务丢失

适用场景：

  • 本地开发调试

  • 单元测试

  • 临时任务（不需要持久化）

SQLite 存储（单机生产）

特点：

  • 持久化存储，重启不丢失

  • 无需额外服务

  • 支持中等规模数据

  • 不支持多节点共享

适用场景：

  • 单机生产部署

  • 需要任务持久化

  • 中小规模任务（< 10万/天）

性能建议：

  • 使用 SSD 存储

  • 定期清理过期任务

  • 考虑分库分表

Redis 存储（分布式）

Redis URL 格式：

特点：

  • 多节点共享任务

  • 支持分布式部署

  • 高性能

  • 需要额外部署 Redis

适用场景：

  • 多节点分布式部署

  • 高并发任务处理

  • 需要任务共享

环境变量配置

环境变量列表

环境变量 对应配置 默认值
NEOTASK_STORAGE_TYPE storage_type memory
NEOTASK_SQLITE_PATH sqlite_path neotask.db
NEOTASK_REDIS_URL redis_url None
NEOTASK_WORKER_CONCURRENCY worker_concurrency 10
NEOTASK_MAX_RETRIES max_retries 3
NEOTASK_RETRY_DELAY retry_delay 1.0
NEOTASK_QUEUE_MAX_SIZE queue_max_size 10000
NEOTASK_LOCK_TYPE lock_type memory
NEOTASK_LOCK_TIMEOUT lock_timeout 30
NEOTASK_ENABLE_METRICS enable_metrics true

配置调优建议

Worker 并发数

任务类型 推荐值 说明
CPU 密集型 CPU 核心数 避免上下文切换
IO 密集型 CPU 核心数 × 2-4 提高并发
混合型 CPU 核心数 × 2 平衡性能

预取大小

场景 推荐值 说明
低延迟要求 1-5 任务快速响应
高吞吐要求 20-50 减少网络往返
任务耗时较长 10-20 平衡内存和吞吐

队列大小

场景 推荐值 说明
内存有限 1000-5000 控制内存使用
标准配置 10000-50000 平衡性能
高吞吐 100000+ 大量任务堆积

超时设置

参数 推荐值 说明
task_timeout 平均耗时 × 3 避免误判超时
lock_timeout 任务耗时 × 2 避免锁提前释放
wait_timeout 任务耗时 × 2 等待结果超时

重试配置

参数 推荐值 说明
max_retries 3-5 避免无限重试
retry_delay 1-5 避免频繁重试

常见配置问题

问题 可能原因 解决方案
任务不执行 Worker 并发数为 0 设置 worker_concurrency >= 1
队列满 queue_max_size 太小 增大队列大小
锁超时 lock_timeout 太小 增加锁超时时间
任务重复 分布式锁配置错误 确保 lock_type="redis"
性能差 Worker 并发不足 增加并发数
