相关链接：GitHub | 官网 | 文档 （架构设计 · 召回策略 · 一致性策略 · RAG 增强）

从 MVP 到专业级剧本分镜智能体的完整演进历程。

项目定位与核心目标

PenShot 是一个剧本分镜智能体系统，将自然语言剧本自动转换为结构化的视频分镜和 AI 提示词，最终对接文生视频模型生成视频片段。

核心目标

目标 说明
零门槛 输入剧本，输出可直接用于视频生成的提示词
高质量 确保分镜的语义准确性、视觉连续性、角色一致性
可扩展 支持超长剧本、多场景、复杂角色关系
可观测 全链路状态追踪、质量审计、人工干预

核心设计原则

原则 说明
渐进增强 每个版本都是独立可用产品，用户按需升级
向后兼容 API 稳定，存储格式平滑升级
可观测优先 关键操作有埋点，问题可追踪
标准驱动 输出格式对齐专业标准（XML/EDL/FCPXML）

核心功能详解

模块 功能 说明
剧本解析 场景识别 从剧本中提取场景边界、地点、时间
角色提取 识别主要角色名称、出场顺序
动作提取 提取关键动作描述
对话提取 提取角色对话内容、情感标注
镜头拆分 镜头类型识别 远景/中景/特写等基础分类
时长估算 基于元素数量估算每镜头时长
镜头序列生成 按时序组织镜头，生成连续ID
提示词生成 文生视频提示词 基础英文提示词格式
负面提示词 默认负面提示词模板
输出 JSON 格式 结构化分镜数据，包含元数据

核心功能详解

质量审计

功能 说明
规则审查 场景数、角色数、时长阈值等硬性规则检查
LLM 深度审查 语义理解层面的质量评估
问题分类 ERROR / MAJOR / MODERATE / WARNING / INFO
审计报告 结构化问题列表，包含位置和建议

问题类型示例

类型 示例 严重程度
场景缺失 未能识别到任何场景 ERROR
角色缺失 对话角色未注册 MAJOR
提示词过短 提示词少于10个单词 WARNING
时长不合理 镜头时长超过8秒 MODERATE
风格不一致 检测到3种以上风格 MODERATE

自动修复

功能 说明
剧本解析修复 补充缺失场景、创建默认角色
分镜生成修复 调整镜头时长、修正镜头类型
提示词修复 补充截断内容、扩展过短提示词
修复历史 记录每次修复操作，支持回滚
修复统计 原始状态 vs 修复后状态对比

修复操作示例

问题 修复策略
没有识别到场景 按段落分割创建默认场景
角色缺失 自动创建缺失角色，补充描述
提示词过短 从原始剧本扩展描述
镜头时长过长 自动裁剪到阈值以内

连续性检查（基础版）

维度 检查内容
角色连续性 角色是否在应有场景中出现
场景连续性 场景切换是否突兀、是否过于频繁
时间连续性 片段间时间间隙、时间重叠

节点失败降级

策略 说明
单节点重试 失败后自动重试（可配置次数）
降级方案 节点失败后切换到备用处理方式
降级日志 记录降级原因和影响范围
优雅退出 节点失败不影响其他节点

错误处理

错误类型 处理策略
网络超时 指数退避重试
LLM 服务错误 切换备用模型或重试
数据校验失败 触发修复流程
资源不足 延迟后重试
系统错误 人工干预

人工干预

功能 说明
干预触发 工作流暂停，等待人工输入
超时处理 超时后自动选择默认决策
决策记录 人工决策历史可追溯
干预恢复 人工决策后工作流继续执行

循环检查

检查项 说明
节点循环限制 单节点最大循环次数（默认3次）
全局循环限制 工作流全局循环上限
循环告警 接近限制时发出警告
循环记录 记录每次循环的原因和结果

片段置信度评分

评分维度 说明
提示词质量 长度、完整性、语义清晰度
时长合理性 是否符合模型限制
角色一致性 角色是否前后一致
场景连续性 与前后片段的衔接
综合评分 0-100 分质量评估

置信度等级

分数范围 等级 说明
90-100 优秀 可直接使用
70-89 良好 建议人工复核
50-69 及格 可能需要修复
0-49 不合格 必须修复

核心功能详解

LangGraph 工作流引擎

能力 说明
状态图编排 节点间依赖关系定义，可视化流程
条件分支 根据决策结果动态路由到不同节点
循环支持 天然支持重试和修复循环
状态持久化 Checkpointer 机制，支持断点保存
节点定义 剧本解析/分镜/分割/转换/审计等10+节点

工作流节点决策

决策状态 说明 后续动作
SUCCESS 节点执行成功 进入下一节点
VALID 验证通过（有小问题） 可继续执行
NEEDS_RETRY 需要重试 重新执行当前节点
NEEDS_REPAIR 需要修复 进入修复流程
NEEDS_HUMAN 需要人工 暂停等待人工决策
FAILED 执行失败 进入错误处理
ABORT 中止流程 结束工作流

三层记忆系统

层级 存储内容 TTL 用途
短期记忆 当前任务上下文、临时修复策略 1小时 单次工作流执行
中期记忆 任务统计信息、修复历史记录 7天 跨任务经验积累
长期记忆 常见问题模式、成功模式库 永久 全局知识沉淀
