NeoTask API 参考

完整的 NeoTask API 文档。

📚 返回首页 | 📖 快速上手 | 📘 API 参考 | ⚙️ 配置说明

TaskPool 类

即时任务池，专注于立即执行的任务。

初始化

参数：

参数 类型 说明
executor Callable 任务执行函数（异步函数）
config TaskPoolConfig 配置对象

示例：

任务提交

submit

提交即时任务（同步）。

参数：

参数 类型 说明
data Dict 任务数据
task_id str 任务ID（可选，自动生成）
priority int/TaskPriority 优先级（0-3，默认2）
delay float 延迟执行时间（秒）
ttl int 任务超时时间（秒）

返回：str - 任务ID

示例：

submit_async

提交即时任务（异步）。

示例：

submit_batch

批量提交任务（同步）。

参数：

参数 类型 说明
tasks List[Dict] 任务数据列表
priority int/TaskPriority 优先级

返回：List[str] - 任务ID列表

示例：

submit_batch_async

批量提交任务（异步）。

任务等待

wait_for_result

同步等待任务完成并返回结果。

参数：

参数 类型 说明
task_id str 任务ID
timeout float 超时时间（秒）

返回：Any - 任务结果

异常：

  • TaskNotFoundError：任务不存在

  • TimeoutError：等待超时

  • Exception：任务执行失败

示例：

wait_for_result_async

异步等待任务完成。

示例：

wait_all

等待所有任务完成（同步）。

参数：

参数 类型 说明
task_ids List[str] 任务ID列表
timeout float 超时时间（秒）

返回：Dict[str, Any] - 任务ID到结果的映射

示例：

wait_all_async

异步等待所有任务完成。

任务查询

get_status

获取任务状态。

返回：Optional[str] - 状态值（pending/running/success/failed/cancelled/scheduled）

示例：

get_status_async

异步获取任务状态。

get_result

获取任务结果。

返回：Optional[Dict] - 包含任务信息和结果

示例：

get_result_async

异步获取任务结果。

get_task

获取完整任务信息。

返回：Optional[Dict] - 包含所有任务字段

get_task_async

异步获取完整任务信息。

task_exists

检查任务是否存在。

返回：bool - 是否存在

任务管理

cancel

取消任务。

返回：bool - 是否成功取消

示例：

cancel_async

异步取消任务。

delete

删除任务。

返回：bool - 是否成功删除

delete_async

异步删除任务。

retry

重试失败的任务。

参数：

参数 类型 说明
task_id str 任务ID
delay float 延迟执行时间（秒）

返回：bool - 是否成功重试

队列控制

pause

暂停处理新任务（已入队的任务继续处理）。

pause_async

异步暂停处理新任务。

resume

恢复处理新任务。

resume_async

异步恢复处理新任务。

clear_queue

清空队列。

clear_queue_async

异步清空队列。

get_queue_size

获取队列大小。

get_queue_size_async

异步获取队列大小。

统计与监控

get_stats

获取实时统计信息。

返回：

get_stats_async

异步获取统计信息。

get_worker_stats

获取 Worker 统计信息。

get_health_status

获取健康状态。

生命周期

start

启动任务池。

shutdown

关闭任务池。

参数：

参数 类型 说明
graceful bool 是否优雅关闭（等待当前任务完成）
timeout float 优雅关闭超时时间

事件回调

on_created

注册任务创建事回调。

示例：

on_started

注册任务开始事件回调。

on_completed

注册任务完成事件回调。

on_failed

注册任务失败事件回调。

on_cancelled

注册任务取消事件回调。

TaskScheduler 类

定时任务调度器，专注于延时/周期任务。

初始化

参数：

参数 类型 说明
executor Callable 任务执行函数
config SchedulerConfig 配置对象

任务提交

submit

提交即时任务。

submit_delayed

延时执行任务。

参数：

参数 类型 说明
data Dict 任务数据
delay_seconds float 延迟秒数
task_id str 任务ID（可选）
priority int/TaskPriority 优先级

示例：

submit_at

指定时间点执行。

参数：

参数 类型 说明
execute_at datetime 执行时间点

示例：

submit_interval

按固定间隔周期执行。

参数：

参数 类型 说明
interval_seconds float 执行间隔（秒）
run_immediately bool 是否立即执行第一次

示例：

submit_cron

按 Cron 表达式周期执行。

参数：

参数 类型 说明
cron_expr str Cron 表达式

Cron 格式：分 时 日 月 周

字段 范围 示例
分钟 0-59 0 表示整点
小时 0-23 9 表示上午9点
日 1-31 * 表示每天
月 1-12 * 表示每月
周 0-6（0=周日） * 表示每周

示例：

周期任务管理

cancel_periodic

取消周期任务。

pause_periodic

暂停周期任务。

resume_periodic

恢复周期任务。

get_periodic_tasks

获取所有周期任务。

返回：

  • wait_for_result(task_id, timeout)

  • wait_for_result_async(task_id, timeout)

  • get_status(task_id)

  • get_result(task_id)

  • cancel(task_id)

  • get_stats()

事件对象

TaskEvent

事件类型：

类型 说明 data 内容
task.created 任务创建 任务字典
task.started 任务开始 {"node_id": "..."}
task.completed 任务完成 {"result": ...}
task.failed 任务失败 {"error": "..."}
task.cancelled 任务取消 None
